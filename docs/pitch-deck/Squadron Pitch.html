/// 04 · THE ANSWER
/// THE SUBSTRATE

a hex grid. a vault. an MCP wire.

topology becomes the permission model, rendered as space.
/// CANVAS · 32×22 · 704 hexes
SPATIAL
hex grid. agents are placeable objects. adjacency is real — you only talk to neighbors.
SHARED
markdown vault. per-agent folder of .md files. obsidian-compatible. coordination state lives as files you can read, edit, branch.
STANDARD
MCP wire. three adjacency-gated tools — send_to, read_neighbor_vault, move_toward. any MCP agent plugs in.
