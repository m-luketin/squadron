squadron /// alpha · invite-only

A spatial multi-agent control plane that runs on your machine.

Agents live on a hex grid. They share a markdown vault, walk to each other, talk through MCP tools. Karpathy-style LLM-Wiki memory. No cloud, no API keys — your existing claude subscription runs the show.

Install (macOS · 60 seconds)
curl -fsSL https://raw.githubusercontent.com/m-luketin/squadron/main/install.sh | bash

/// what it does

  • Hex world. Spawn agents, drag them around, paint walls + routers.
  • Memory graph. Each agent owns a markdown wiki. Karpathy LLM-Wiki layout by default.
  • Walking. Agents pathfind around walls + each other to start a conversation.
  • Skills. Install markdown capabilities from a starter library or any GitHub URL.
  • Inter-agent comms. send_to / read_neighbor_vault / move_toward via MCP.
  • Subscription billing. Uses your Pro/Max OAuth — no API-key surprise charges.
  • Persistent. SQLite world; resume across restarts.
  • OSS. All your data stays local. No telemetry beyond opt-in.

/// requirements

macOS Bun 1.3+ claude-code CLI signed in cloudflared (optional, remote access)

The installer above checks each of these and installs anything missing. You'll be prompted to claude auth login mid-install if you aren't already signed in.
